# 소개실1 기말 프로젝트

## 📋 프로젝트 소개
이 프로젝트는 소개실1 과목의 기말 프로젝트입니다.

## 🚀 파이썬으로 빠른 실행하기

### 필요 조건
- Python 3.x

### 설치 및 실행 방법
1. 프로젝트를 클론합니다:
```bash
git clone [repository-url]
```

2. 프로젝트 디렉토리로 이동:
```bash
cd [project-directory]
```

3. 서버 실행:
```bash
python -m http.server 8000
```

4. 웹 브라우저에서 다음 주소로 접속:
```
http://localhost:8000
```

## 🛠️ 기술 스택
- HTML/CSS/JavaScript
